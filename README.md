# EDU circuit (Sebastian Viktorik)

Statische Website mit interaktiven Lernmodulen für elektronische Schaltungen –
nachgebaut und gestalterisch verbessert nach [edu-circuit.at](https://edu-circuit.at/).
Inhalte, Texte und Formeln entsprechen dem Original; verbessert wurden ausschließlich
Design, Layout und Scroll-Effekte.

## Seiten
- `index.html` – Übersicht mit Schaltungsmodulen
- `Spannungsteiler.html`, `Stromteiler.html`, `Ueberlagerungsprinzip.html`, `Versorgung.html` – Module mit interaktiven Rechnern
- `kontakt.html`, `impressum.html`, `datenschutz.html`

## Struktur
- `css/style.css` – Design-System
- `js/main.js` – Navigation, Scroll-Effekte und Rechner
- `Bilder/` – Logo und Schaltungs-Grafiken (SVG)

## Lokal öffnen
Einfach `index.html` im Browser öffnen – kein Build, kein Server nötig.
